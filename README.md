# Diagnostic PC

Outil simple pour diagnostiquer un PC Windows.

## Installation

1. Télécharger la dernière version :
👉 https://github.com/Pinkywhisky/diagnostic-pc/releases/latest

2. Lancer le setup

3. Cliquer sur :
- Lancer le diagnostic
- Envoyer par mail

## Important

Si Windows affiche un message de sécurité :
→ cliquer sur "Informations complémentaires"
→ puis "Exécuter quand même"

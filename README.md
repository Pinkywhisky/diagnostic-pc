# Diagnostic PC

Outil simple pour diagnostiquer un PC Windows et envoyer les résultats par mail.

---

## 📥 Installation

1. Télécharger la dernière version :
👉 https://github.com/Pinkywhisky/diagnostic-pc/releases/latest

2. Lancer `Setup_Diagnostic_PC.exe`

3. Suivre les étapes d’installation

---

## 🚀 Utilisation

1. Lancer l'application
2. Cliquer sur **Lancer le diagnostic**
3. Cliquer sur **Envoyer par mail**

---

## ✨ Fonctionnalités

- Vérification du système (OS, uptime, BIOS)
- Analyse réseau (passerelle dynamique, Internet, DNS)
- Test des ports (80 / 443)
- Analyse CPU / RAM
- Vérification des disques (logiques et physiques)
- Vérification sécurité (Defender, Firewall)
- Vérification de services Windows critiques
- Génération d’un rapport clair avec résumé (OK / WARNING / CRITICAL)
- Envoi du rapport par mail

---

## 🔄 Mise à jour

L’application vérifie automatiquement les mises à jour au démarrage.

- Si une nouvelle version est disponible, une fenêtre s’affiche
- Il suffit de cliquer sur **Oui** pour lancer l’installation

👉 Aucun bouton manuel n’est nécessaire

---

## ⚠️ Avertissement de sécurité Windows

Si Windows affiche un message :

- Cliquer sur **Informations complémentaires**
- Puis sur **Exécuter quand même**

(Ce message est normal pour une application non signée)

---

## 🧠 À savoir

- L’outil fonctionne sans configuration
- Aucun accès administrateur requis (dans la majorité des cas)
- Les tests réseau utilisent la configuration réelle du PC (passerelle automatique)

---

## 📬 Support

En cas de problème, vous pouvez :
- Relancer un diagnostic
- Réinstaller l’application avec la dernière version
